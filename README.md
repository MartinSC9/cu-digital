# My Portfolio

This project is a personal portfolio developed with React. It showcases my technical skills, work experience, and education. It is designed to be a visual representation of my capabilities as a Fullstack developer specialized in React and .NET.

## Table of Contents

- [Features](#features)
- [Technologies Used](#technologies-used)

## Features

- Showcases technical skills in Frontend and Backend.
- Work experience section with highlighted achievements.
- Educational information.
- Option to schedule meetings.

## Technologies Used

- **Frontend:**
  - React
  - React Router
  - React Icons
- **Backend:**
  - .NET
- **Database:**
  - MySQL
- **DevOps:**
  - GitHub Actions
